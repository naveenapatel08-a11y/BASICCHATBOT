# BASICCHATBOT

## Description
A simple chatbot built using Python that responds to user inputs with predefined answers.

## Features
- Interactive conversation
- Predefined responses
- Easy to use
- Beginner-friendly project

## Technologies Used
- Python

## How to Run
1. Download the project files
2. Open terminal or command prompt
3. Run:

python chatbot.py
